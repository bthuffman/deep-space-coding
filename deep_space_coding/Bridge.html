USS Avalon: Main Bridge

The purpose of this website is to practice the elements of full stack web development by recreating the styles and structures found on a Star Fleet space ship's computer controls. To this end I present Deep Space Coders USS Avalon. See the table below for a breakdown of the various ship controls available aboard the starship Avalon.

Control Section Description Commanding Officer
Tactical Deals with strategic resources, financial assets, and financial planning. VACANT
